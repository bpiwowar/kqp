Kernel Quantum Probability project
Author:
B. Piwowarski
Date:
May 2011

This project aims at providing an API that allows to compute quantum densities (semi-definite positive hermitian operators) or events (subspaces). It provides tools to compute quantum probabilities and update densities (conditionalisation), and supports the use of kernels to implicitely define the space, thus allowing working in very high dimensional spaces.

This project is hosted on Github.
