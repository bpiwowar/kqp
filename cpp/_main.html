There are three main modules:

  • Feature matrices are used to represent a set of feature vector. In a finite vectorial space, this is typically a matrix.
  • Operator builders are the classes that compute a thin representations of (kernel) linear operators, based on a single type of feature matrix;
  • Probabilities can then be computed from built operators (they both define events and quantum probability densities).
